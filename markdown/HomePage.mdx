# Qarakey haqqında

Bul sayt **á**, **ú**, **ń**, ... sıyaqlı háriplerdi teriwge kómek beretuǵın **klaviatura**nı júklep alıw ushın jaratılǵan:

**Qarakey — Qaraqalpaqsha klaviatura maketleri.**

Onı qaytıp ornatıw, paydalanıw sıyaqlı maǵlıwmatlardı tómendegi siltemelerdi basıw arqalı yáki menyu bólimindegi siltemelerdi basıw arqalı bilip alıwıńız múmkin:

- [`/use`](/use) — paydalanıw
- [`/install`](/install) — ornatıw
