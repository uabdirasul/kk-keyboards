Bul klaviaturanı ornatıwdan aldın kompyuterińiz qanday ekenligin bilip alıwıńız kerek.

Házirshe bul klaviatura tek Windows kompyuterleri ushın. Eger sizde Windows bolsa tómendegi basqıshlar arqalı onı júklep alıwıńız múmkin:

1. Siziń kompyuterińiz x32 **bit**lik pe yáki x64 **bit**lik pe? Onı bilip alıw ushın:

   - **Проводник** dástúrin ashıń. Onı `Win + E` túymeleri arqalı tezirek ashıwıńız múmkin.
   - Payda bolǵan ayna sol dástúrge tiyisli, ol jerden **Этот компьютер** di tabıń. Ol shep tárepte dástúrdiń orta bólimlerde jaylasqan.
   - **Этот компьютер** ústine **мышка**nı alıp barıp, onıń oń tárepin basıń. Payda bolǵan aynadan **Свойства**nı saylań.
   - Tómendegidey ayna shıǵadı. Ol jerde System Typeti tawıp 64 yáki 32 bit ekenligin anıqlawıńız múmkin:

       src="https://i.ibb.co/7pMdHLw/Screenshot-from-2024-11-08-19-23-08.png"
       alt="My Image"
       width={800}
       height={600}
     />

2. Neshe **bit** ekenligin bilip bolǵannan keyin, usı bette, joqarıdaǵı x32, x64, x86 lardan ózińizge keregin saylap, júklep alıwıńız múmkin.
3. Júklep alınǵan fayldı ashın hám **Next** túymesin izbe-iz basıń. Nátiyjede klaviatura kompyuterińizge ornatıladı.
4. Paydalanıwdı baslaw ushın usı siltemege basıń: [`/use`](/use)
